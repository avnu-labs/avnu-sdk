# AVNU SDK Example

A minimal Next.js application demonstrating all features of the [AVNU SDK](https://github.com/avnu-labs/avnu-sdk) for Starknet.

## Features Demonstrated

- **Swap** - Token exchange with route optimization
- **DCA** - Dollar Cost Averaging orders
- **Staking** - AVNU stake delegation
- **Market Data** - Prices, volumes, TVL feeds
- **Tokens** - Token information and search
- **Paymaster** - Gasless transactions

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- AVNU SDK
- starknet.js
